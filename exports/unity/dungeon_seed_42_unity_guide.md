## How to Import This JSON

### Option A — JsonUtility (built-in, zero extra packages)

> ⚠️ `JsonUtility` does not support `null` object references.
> If `lastPath` is absent from the JSON, leave this field out of the struct
> or use Option B instead.

1. Copy the `.json` file into `Assets/Resources/` (no extension in the load call).
2. Load and deserialize:

```csharp
TextAsset jsonAsset = Resources.Load<TextAsset>("dungeon_seed_42");
UnityMapData data   = JsonUtility.FromJson<UnityMapData>(jsonAsset.text);
```

### Option B — Newtonsoft.Json (recommended)

Install via **Window → Package Manager → Add by name**: `com.unity.nuget.newtonsoft-json`

```csharp
using Newtonsoft.Json;

TextAsset jsonAsset = Resources.Load<TextAsset>("dungeon_seed_42");
UnityMapData data   = JsonConvert.DeserializeObject<UnityMapData>(jsonAsset.text);
```

### Option C — StreamingAssets (runtime file access)

```csharp
string filePath = Path.Combine(Application.streamingAssetsPath, "dungeon_seed_42.json");
string json     = File.ReadAllText(filePath);
UnityMapData data = JsonConvert.DeserializeObject<UnityMapData>(json);
```

## Example MonoBehaviour

Save as `LoadMap.cs` anywhere inside `Assets/`:

```csharp
using UnityEngine;
using UnityEngine.Tilemaps;
using Newtonsoft.Json;

public class LoadMap : MonoBehaviour
{
    [Header("Data")]
    [SerializeField] private TextAsset mapJson;

    [Header("Tilemap")]
    [SerializeField] private Tilemap tilemap;

    [Header("Tile Assets")]
    [SerializeField] private TileBase floorTile;
    [SerializeField] private TileBase wallTile;
    [SerializeField] private TileBase doorTile;
    [SerializeField] private TileBase corridorTile;

    private UnityMapData    _data;
    private UnityTileData[,] _grid;  // [x, y] for O(1) lookup

    void Start()
    {
        // 1. Deserialise JSON
        _data = JsonConvert.DeserializeObject<UnityMapData>(mapJson.text);

        // 2. Build random-access 2D grid from flat array
        //    Flat storage: index = tile.y * _data.width + tile.x
        _grid = new UnityTileData[_data.width, _data.height];
        foreach (var tile in _data.tiles)
            _grid[tile.x, tile.y] = tile;

        // 3. Paint the Tilemap
        //    Unity's Y axis points up, so negate Y to flip the map right-side up.
        foreach (var tile in _data.tiles)
        {
            TileBase tb = ResolveTile(tile.tileType);
            if (tb != null)
                tilemap.SetTile(new Vector3Int(tile.x, -tile.y, 0), tb);
        }

        // 4. Visualise A* path (if present)
        if (_data.lastPath?.waypoints is { Length: > 1 })
        {
            var wps = _data.lastPath.waypoints;
            for (int i = 0; i < wps.Length - 1; i++)
                Debug.DrawLine(
                    new Vector3(wps[i].x + 0.5f,   -wps[i].y   - 0.5f, 0),
                    new Vector3(wps[i+1].x + 0.5f, -wps[i+1].y - 0.5f, 0),
                    Color.magenta, duration: 60f);
        }

        Debug.Log($"Loaded {_data.generatorType} map — "
                + $"{_data.width}x{_data.height}, {_data.rooms.Length} rooms, seed {_data.seed}");
    }

    // Map tileType string → TileBase asset
    TileBase ResolveTile(string tileType) => tileType switch
    {
        "Floor"    => floorTile,
        "Wall"     => wallTile,
        "Door"     => doorTile,
        "Corridor" => corridorTile,
        _          => null
    };

    // Helper: world-space tile query
    public bool TryGetTile(int x, int y, out UnityTileData tile)
    {
        tile = default;
        if (x < 0 || x >= _data.width || y < 0 || y >= _data.height) return false;
        tile = _grid[x, y];
        return true;
    }
}
```

## Tilemap Integration Tips

**Batch painting** — For large maps, replace the per-tile `SetTile` loop with a
single `Tilemap.SetTilesBlock(BoundsInt, TileBase[])` call. Build a `TileBase[]`
in the same row-major order and pass `new BoundsInt(0, -height+1, 0, width, height, 1)`
as the bounds (accounting for the Y-flip).

**Colliders** — Add a `TilemapCollider2D` + `CompositeCollider2D` to the Wall
layer and set the tilemap's Collision Type to `Sprite`. Non-walkable tiles
(`isWalkable == false`) will automatically generate merged colliders.

**Rule Tiles** — Replace simple `TileBase` assets with Unity's `RuleTile`
(from the 2D Extras package) to get automatic wall auto-tiling based on
neighbour adjacency — no code changes required.
